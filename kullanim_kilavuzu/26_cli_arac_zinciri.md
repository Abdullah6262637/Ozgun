# CLI (Komut Satırı Arayüzü) Kullanım Kılavuzu

`oz-cli` araç zincirinin tüm komutları ve parametre detayları.

## Komut Listesi
1. **`yeni <isim>`**: Belirtilen isimde şablon bir proje klasörü oluşturur.
2. **`derle`**: Projeyi derler ve hata kontrolü yapar.
3. **`derle --yerel`**: Projeyi C koduna transpile edip yerel `.exe` derlemesi yapar.
4. **`calistir`**: Projeyi doğrudan derleyip VM üzerinde çalıştırır.
5. **`test`**: Entegrasyon testlerini koşturur.
